# 기초 통계 개념


통계의 본질은 "분산의 마법" 입니다. 
자료의 모습 중 대표적인 특징을 잡아서 설명,
그것이 자료의 대표값입니다. 
- 대표값 : 평균, 중간값, 최빈값, 표준편차, 구간, 최소값...

### 평균
평균은 누구나 알지만 어떤 특징이 있을까요?
- 평균의 의미: 자료의 중심값으로서 자료의 특성을 대표하는 값
- 수학적 연산이 쉬우며, 모든자료의 영향을 받아 이상한 값까지 영향을 받는다.
- 분산(표준편차)의 계산 등 통계 분석의 대표값으로 사용 

### 표준편차
분산(표준편차)는 무엇일까?

예 : {1,2,3,4,5} 에서 평균은 3이고,


![img](https://user-images.githubusercontent.com/45762604/82795051-65583880-9eae-11ea-9776-98d528533057.png)


- 데이터 평균값을 중심으로 퍼져 있는 평균적인 거리
- 수식의 분자 부분은 각 값에서 평균을 뺸것입니다.
- n(자료의 개수 : 5)가 아닌 n-1 (4)로 나누는 이유는 이미 분자에서 평균값으로 한번씩 다 빼 주었기 때문입니다.

- 분산  = 제곱합 / 자유도  = 제곱합의 평균

$$표준편차 = \sqrt 분산$$ 

### 왜 평균과 분산(표준편차)인가?
- 중심값으로 평균 말고도 중간값이나 최빈값 등 여러 대표값이 있지만 평균과 분산(표준편차)인 이유는 계산이 쉽다
- 가우스에 따르면, 데이터 불규칙성이 정규 분포를 따르고 있으며 최소제곱법이 가장 추정방법이고, 그결과 평균값이 가장 좋은 추정값이 된다
- 또한 체비체프에 따르면, 데이터 불규칙성이 어떠하든 범위안에는 반드시 전체 3/4 이상의 데이터가 존재가 증명됨


### 유의 수준 p-값
예 : p-값이 0.05보다 작으니 유의하다
도대체 p-값은 무엇이고, 0.05는 무엇이며, 유의하다는건 무엇일까?
- p-value : probability value -> 확률값 
- 무엇에 대한 확률값?? -> 어떤 사건의 우연히 발생할 확률이 바로  p-값이다
- p-값이 가지수있는 값은 0~1이며, 0.05는 %로 바꾸면 5%확률입니다.
즉 0.05보다 작은건 어떤 사건이 우연히 발생할 확률이 5%보다 작다는 의미입니다.
- 만약 p-값이 0.05보다 크면, 이사건은 우연히 발생한 것이고, 인과관계가 없다고 판단하는 것입니다.


### 통계적 가설
귀무가설, 대립가설
- 귀무가설 : 어떤 사건이 우연히 일어날 확률을 구할때 p-값을 사용하는데 그 사건이 우연히 일어났다 라고 가정하는것
- 대립가설 : 그 사건이 우연히 일어난것이 아니고 뭔가 이유가 있다고 가정하는것 

- 가설을 선택하는 것은 p-값에 달렸습니다. 
만약 0.05보다 작다면 대립가설을 선택하고, 
크다면 귀무가설을 선택하는 것입니다.


### 변수란?
변수 : 하나의 개념을 대표하는 상징으로서 그것의 특성이 갖는 값이나 강도, 크기의 차이  -> 변하는 숫자

예) 고객만족도는 고객마다 다르기 때문에 값이 변화한다.


### 변수의 종류
- 연속 변수
	- 연속적인 값을 가지는 변수 
	- 예) 나이, 점수, 무게, 가격 등
- 범주변수(이산변수)
	- 서로 다른 것으로 구분되는 변수
	- 예) 성별(남자/여자), 애완동물(강아지/고양이)

### 척도
척도(scale): 측정된 변수의 값을 표현하는 수준(levels of measurement)을 의미
- 명목척도(nominal scale) -> 순위가 없다, 할당된 값은 의미가 없다 - 성별/인종
- 서열척도(ordinal scale) -> 순위가 있다 - 성적/학력/경제수준
- 구간척도(interval scale) -> 범주사이 등간성 있다, 할당된 값은 임의 단위로 의미없음 
(덧셈은 가능하나 곱셈은 안됨) - 온도
- 비율척도(ratio scale) -> 범주사이 등간성 있다, 할당된 값은 임의 단위로 비율과 절대 'o' 의미 있음(덧셈,곱셈은 안됨) - 키/몸무게/나이/시간

### 척도의 중요성
- 척도에 따라 적용가능한 통계 분석방법이 다름
- 숫자로 표현된 경우라 하더라도 무조건 사칙연산이 가능하지는 않음. 그 숫자의 의미(=척도)를 이해해야
	- 예) 남자 = 1, 여자 = 2로 표현하는 경우
	- 수로 표현되었지만 1+1=2 와 같이 계산하면 남자 2명이 여자 1명과 같다는 이상한 해석이 됨
- 가능하면 비율척도나 등간척도의 형태로 자료 수집을 하는 것이 분석에 용이
	- 서열척도로 조사: "어린이, 청장년, 노인"와 같이 나누어 조사
	- 비율척도로 조사: 만 나이로 조사
	- 비율척도는 다양한 계산과 분석이 가능하지만 서열척도는 어린이 몇 명, 노인 몇 명 등의 분석만 가능


### 상관관계와 상관계수

- 상관관계 : 한 변수와 다른 변수가 공변하는 함수관계
예) 	y = wx + b
- 상관계수 : -1 에서 +1, 
계수의 +/-는 방향의미, 
크기는 힘의 의미 



### 변산성 측정치 (분산, 표준편차, 범위, 사분위간 범위)

- 변산성(variability): 자료가 흩어져 있는 정도, 혹은 개체에 따라 변할 수 있는 정도

- 범위
	- 자료가 갖는 최대값과 최소값 사이의 거리, 즉 자료가 얼마나 퍼져있는가를 나타냄
	- 범위 = 최댓값 - 최솟값
- 분산
- 표준편차
- 사분위간 범위
	- 사분위간 범위(IQR, InterQuartile Range)는 제3사분위수에서 제1사분위수 간의 범위
	- 사분위수란 전체 데이터를 작은 값부터 큰 값까지 순서대로 나열한 후 4등분 하였을 때, 각 지점에 해당하는 값
		- 제1사분위수(Q1): 25% 지점
		- 제2사분위수(Q2): 50% 지점 = 중간값
		- 제3사분위수(Q3): 75%
	

### 공분산
- 공분산(covariance): 두 변수가 함께 변화하는 정도를 나타내는 지표

- 공분산의 해석 
	- 부호 : 
		- 공분산이 +인 경우: 두 변수가 같은 방향으로 변화(하나가 증가하면 다른 하나도 증가)
		- 공분산이 -인 경우: 두 변수가 반대 방향으로 변화(하나가 증가하면 다른 하나는 감소)
  - 크기 :
		- 공분산 = 0이면 두 변수가 독립, 즉, 한 변수의 변화로 다른 변수의 변화를 예측하지 못함
		- 공분산의 크기가 클 수록 두 변수는 함께 많이 변화
